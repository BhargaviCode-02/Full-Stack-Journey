Introduction_to_C_plus_plus

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C language.

It supports multiple programming paradigms including procedural, object-oriented, and generic programming.

C++ is widely used for system programming, game development, and high-performance applications.

  • High performance and efficiency
  • Strong memory management capabilities
  • Rich standard library (STL)
  • Multi-paradigm support
Variables and Data Types

C++ supports several fundamental data types including int, float, double, and char.

Variables must be declared with a type before use: int age = 25;

Boolean values use bool type with true and false literals.

int main() { int x = 10; double pi = 3.14159; return 0; }

Strings are handled via #include <string> and std::string.

Control Structures

C++ provides if, else if, else for conditional execution.

Switch statements handle multiple discrete conditions efficiently.

  • for loops for known iteration counts
  • while loops for condition-based iteration
  • do-while loops execute at least once

Range-based for loops (C++11): for(auto& item : container)

Functions

Functions promote code reusability and modularity. Declaration syntax: returnType functionName(parameters);

Pass-by-value is default; pass-by-reference uses & modifier.

void swap(int& a, int& b) { int temp = a; a = b; b = temp; }

Function overloading allows multiple functions with same name but different parameters.

Default parameters: int add(int a, int b = 0)

Pointers and References

Pointers store memory addresses: int* ptr = &variable;

References are aliases: int& ref = variable;

  • nullptr for null pointers (C++11)
  • Pointer arithmetic for array traversal
  • Smart pointers (unique_ptr, shared_ptr) for memory safety

Dereference operator * accesses pointed-to value.
